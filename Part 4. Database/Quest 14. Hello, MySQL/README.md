## Checklist
* RDBMS에 길이를 알 수 없는 배열을 저장하려면 어떻게 설계해야 할까요?
* RDBMS 테이블의 정규화는 무엇인가요?
* MySQL 엔진에는 어떤 것들이 있나요?
* RDBMS에서 테이블의 인덱싱은 무엇인가요? 인덱싱을 하면 어떤 점이 다르며, 어떤 식으로 동작하나요?
